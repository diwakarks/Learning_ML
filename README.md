# Learning ML

A collection of Machine Learning projects built while learning Data Science and Machine Learning using Python and Scikit-Learn.

This repository documents my hands-on learning journey, covering data preprocessing, visualization, regression, classification, clustering, dimensionality reduction, and model evaluation techniques.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn
* Jupyter Notebook

## Projects

### 1. Titanic Survival Prediction

* Classification Project
* Predicts passenger survival on the Titanic.
* Concepts: Missing Value Handling, Data Cleaning, Exploratory Data Analysis, Logistic Regression.

### 2. Heart Disease Prediction

* Classification Project
* Predicts the presence of heart disease using patient medical data.
* Concepts: Logistic Regression, Confusion Matrix, Accuracy Evaluation.

## Machine Learning Topics Covered

* Data Cleaning
* Data Visualization
* Feature Selection
* Train-Test Split
* Linear Regression
* Logistic Regression
* K-Nearest Neighbors (KNN)
* Decision Trees
* Random Forest
* K-Means Clustering
* Principal Component Analysis (PCA)
* Confusion Matrix
* Accuracy, Precision, Recall and F1 Score

This repository serves as a record of my Machine Learning learning journey. New projects, algorithms, and experiments will continue to be added as I progress toward AI.
