# Heart Disease Prediction

## Overview

This project predicts whether a person has heart disease based on various medical attributes using machine learning techniques.

## Dataset

The dataset contains patient health information such as:

* Age
* Sex
* Chest Pain Type
* Resting Blood Pressure
* Cholesterol
* Fasting Blood Sugar
* Resting ECG Results
* Maximum Heart Rate Achieved
* Exercise Induced Angina
* ST Depression
* Slope of ST Segment
* Number of Major Vessels
* Thalassemia

Target Variable:

* Heart Disease (0 = No Disease, 1 = Disease)

## Project Workflow

1. Data Loading
2. Data Exploration

5. Model Training
6. Model Evaluation
7. Prediction
